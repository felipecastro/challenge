# Challenge for Backend Developer

Technologies used:

- Node;
- Express;
- Mocha;
- Chai;

# How to run

npm install --save-dev nodemon

npm install express --save

npm run start

# How to test

At Postman : 
http://localhost:3000/products
http://localhost:3000/products/vegan
http://localhost:3000/products/vegan?order=asc

With Mocha :

npm run test
